Start prediction

To start using PON-P3, there are two possible input methods: one via a text box and the other by uploading a file. The platform offers three interfaces for processing data:

  1. Protein Prediction
  2. Transcript Variation
  3. Genomic Variation

Each of these interfaces requires inputs to be in a specific format, which is provided alongside every input field for guidance. After processing, you will need to enter your email address to receive the results. The results will be sent to you in both CSV and PDF formats.

Input Protein Sequence

You can input data via the text box by first entering refseq_ids followed by a ">" symbol. After that, you can enter multiple variation_ids on new lines. For uploading a file, the file must contain two columns with header names as refseq_ids and variation_ids. Please refer to the format example below:

Example

  1. Protein Sequence Variation
    2. >NP_000005.3
M1A
M2A
>NP_000005.3
M1A
M2A

Input Transcript Variation

To start predicting transcript variations, you have two options:

  • Via input (text field): You can click on the example to see the format: NM_000016.6:c.613G>C. It must start with "NM", and there should be a ">" before the character, followed by only one character.
  • Via file upload: The file must contain only one column without a header.

Example

  1. Input Transcript Variation
    2. NM_000016.6:c.613G>C
NM_000016.6:c.653C>G
NM_000017.4:c.319C>T

After entering email you will receive email of text file in the format of csv and pdf file

Input Genomic Variation

To start predicting genomic variations, you have two options:

  • Via input (text field): You can click on the example to see the format: chr1:g.75745819G>C. It must start with "chr", and there should be a ">" before the character, followed by only one character.
  • Via file upload: The file must contain only one column without a header.

Example

  1. Input Genomic Variation
    2. chr1:g.75745819G>C
chr2:g.47789620T>A
chr3:g.12156789A>G

After entering your email, you will receive a text file in CSV format and a PDF file via email.
